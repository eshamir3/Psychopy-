Instructions

{% if experiment.name == 'Emotional Stroop Task' %}
  • Words will appear in different colors (red, blue, green, yellow)
  • Press the key that matches the COLOR of the word:
    • RED = R key
    • BLUE = B key
    • GREEN = G key
    • YELLOW = Y key
  • Ignore what the word says - focus only on its color
{% elif experiment.name == 'N-Back Task' %}
  • Letters will appear one at a time
  • Press SPACE when the current letter matches the one from N items back
  • For example, in 2-back:
    • If you see: A → B → A
    • Press SPACE when the second A appears
{% elif experiment.name == 'Go/No-Go Task' %}
  • Letters will appear one at a time
  • Press SPACE for target letters (e.g., X)
  • Do NOT press any key for non-target letters
  • Respond as quickly as possible while maintaining accuracy
{% endif %}
