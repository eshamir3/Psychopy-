Experiment User Manuals

Emotional Stroop Task

The Emotional Stroop Task measures attention and emotional processing by having participants identify the color of emotionally charged words.

How to Play:

  • Words will appear in different colors (red, blue, green, yellow)
  • Press the key that matches the COLOR of the word:
    • RED = R key
    • BLUE = B key
    • GREEN = G key
    • YELLOW = Y key
  • Ignore what the word says - focus only on its color

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Flanker Task

The Flanker Task measures attention and response inhibition by having participants respond to the direction of a central arrow while ignoring surrounding arrows.

How to Play:

  • Look at the MIDDLE arrow in each display
  • Press LEFT arrow key if the middle arrow points left
  • Press RIGHT arrow key if the middle arrow points right
  • Ignore the arrows on the sides

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Go/No-Go Task

The Go/No-Go Task measures response inhibition by having participants respond to certain letters and withhold responses to others.

How to Play:

  • Letters will appear one at a time
  • Press SPACEBAR for X, Y, or Z
  • Do NOT press for A, B, or C

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

N-Back Task

The N-Back Task measures working memory by having participants identify when a current stimulus matches one from N items back.

How to Play:

  • Letters will appear one at a time
  • Press SPACEBAR if the letter matches the one from N letters ago
  • Do NOT press if it's different
  • For example, with 2-back:
    • A → B → A → (press SPACEBAR!)
    • A → B → C → (don't press)

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Choice Reaction Time Task

The Choice Reaction Time Task measures basic processing speed and decision making by having participants respond to different shapes with corresponding number keys.

How to Play:

  • Different shapes will appear on screen
  • Press the number key that matches the shape:
    • ★ = 1
    • ● = 2
    • ■ = 3
    • ▲ = 4

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Consonant Trigram Task

The Consonant Trigram Task tests short-term memory by requiring you to recall three letters after performing a distraction task.

How to Play:

  • Three capital letters (e.g., DKT) will briefly appear on screen
  • Try to memorize them in the order shown
  • You will then see a number (e.g., 300) and be asked to count backwards by a fixed amount (e.g., by 3s)
  • Count in your head while numbers appear (e.g., 297, 294, ...)
  • When prompted, type the original letters in the order you saw them

Important Notes:

  • Accuracy matters — try to recall all three letters correctly
  • You’ll have limited time to type your response
  • Practice trials are available before the real task

Value Choice Task

The Value Choice Task measures decision-making preferences by asking you to choose between two different reward options.

How to Play:

  • You will see two gray cards on screen, each with a different scenario or reward
  • Click the card that reflects the choice you personally prefer
  • There are no right or wrong answers — go with your instinct
  • You’ll receive feedback after each choice

Important Notes:

  • Try to read and respond within the time limit
  • Card sides (left/right) may be randomized for each trial
  • Practice is available before the main trials

Immediate & Delayed Memory Task (IMT/DMT)

The IMT/DMT measures working memory and impulsivity by having you compare two 5-digit numbers in sequence. Sometimes you’ll see a brief “distractor” phase before the second number (in DMT), designed to interrupt your memory.

How to Play:

  • You will see one 5-digit number (the “previous” stimulus) for a fixed duration.
  • In IMT (Immediate Memory Task), the next number appears immediately.
  • In DMT (Delayed Memory Task), if the first number repeats exactly, you’ll see 3 identical distractor numbers (“12345”) before the second target appears.
  • After that, you will see the second 5-digit number.
  • Press S if the two numbers are the same, or D if they are different.
  • You have up to 5 seconds to respond each trial; if you don’t respond in time, you’ll see a warning.
  • Practice rounds are provided before the main experiment to familiarize you with both IMT and DMT.

Important Notes:

  • Targets (exact repeats) only trigger the distractor phase in DMT—non-matching trials go straight to the response prompt.
  • Distractors are always the same 5-digit string (“12345”), repeated three times to impose a delay.
  • Make sure you understand the difference between IMT and DMT before starting the main trials.
  • Your reaction time (RT) and accuracy are recorded on every trial.
  • The experiment includes separate counts for IMT and DMT practice, and for IMT and DMT main trials.
