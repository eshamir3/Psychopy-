Experiment User Manuals

Emotional Stroop Task

The Emotional Stroop Task measures attention and emotional processing by having participants identify the color of emotionally charged words.

How to Play:

  • Words will appear in different colors (red, blue, green, yellow)
  • Press the key that matches the COLOR of the word:
    • RED = R key
    • BLUE = B key
    • GREEN = G key
    • YELLOW = Y key
  • Ignore what the word says - focus only on its color

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Flanker Task

The Flanker Task measures attention and response inhibition by having participants respond to the direction of a central arrow while ignoring surrounding arrows.

How to Play:

  • Look at the MIDDLE arrow in each display
  • Press LEFT arrow key if the middle arrow points left
  • Press RIGHT arrow key if the middle arrow points right
  • Ignore the arrows on the sides

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Go/No-Go Task

The Go/No-Go Task measures response inhibition by having participants respond to certain letters and withhold responses to others.

How to Play:

  • Letters will appear one at a time
  • Press SPACEBAR for X, Y, or Z
  • Do NOT press for A, B, or C

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

N-Back Task

The N-Back Task measures working memory by having participants identify when a current stimulus matches one from N items back.

How to Play:

  • Letters will appear one at a time
  • Press SPACEBAR if the letter matches the one from N letters ago
  • Do NOT press if it's different
  • For example, with 2-back:
    • A → B → A → (press SPACEBAR!)
    • A → B → C → (don't press)

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment

Choice Reaction Time Task

The Choice Reaction Time Task measures basic processing speed and decision making by having participants respond to different shapes with corresponding number keys.

How to Play:

  • Different shapes will appear on screen
  • Press the number key that matches the shape:
    • ★ = 1
    • ● = 2
    • ■ = 3
    • ▲ = 4

Important Notes:

  • The task includes practice trials to help you get used to the format
  • Try to respond as quickly and accurately as possible
  • You can press ESC at any time to exit the experiment
